# Yukti Distribution Policy

## Canonical Source
- GitHub repository is the authoritative scientific source.
- Contains full source code, tests, build scripts, and documentation.

## Binary Distribution
- No prebuilt binaries or DMG files are distributed via GitHub.
- The only official binary distribution is via the Apple Mac App Store.

## Rationale
- Ensures supply-chain integrity
- Ensures Apple notarization and sandbox compliance
- Preserves open-science reproducibility

## Developer Freedom
- Developers may build binaries locally using provided build scripts.
- Redistribution of binaries is not endorsed by the Yukti project.
